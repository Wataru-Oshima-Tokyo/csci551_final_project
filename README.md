./astar_waypoints

you can choose below.

100x100. 
200x200. 
300x300. 
400x400. 
401x400. 
500x400. 
500x450. 
501x400     
600x600. 
1000x1000. 
2000x2000. 

## known issue
if you increase the size of the map more than 200x200, the thread gives you a segfault error.
