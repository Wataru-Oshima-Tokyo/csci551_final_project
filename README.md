200x200<br>
300x300<br>
400x400<br>
401x400<br>
500x400<br>
500x450<br>
501x400<br> 
600x600<br>
1000x1000<br>
2000x2000<br>

if you increase the size of the map more than 200x200, the thread gives you a segfault error.
